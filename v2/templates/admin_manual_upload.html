Manual CSV upload

Run the pipeline from CSV exports instead of the GA4/GSC API. Use this when the service account hasn't been added to the client's GA4 property yet, or for one-off audits. At least one source required — more sources mean richer analysis.

recommended

Export from GA4 → Explore → your report → Share icon → Download CSV. Include columns: Page path, Sessions, Total users, and ideally Engaged sessions, Average engagement time, Event count, Conversions.

What happens next
  1. We parse each CSV and normalise the data into the same shape the API path produces.
  2. You'll see a preview screen with row counts, top pages, detected date range. Confirm or correct the date range there.
  3. Hit Run pipeline on the preview screen to kick off the five-agent analysis on the parsed data.
